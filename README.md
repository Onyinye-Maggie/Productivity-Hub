Productivity Hub 🚀

A modern React productivity dashboard that helps users manage tasks, track expenses, organize study plans, monitor job applications, and build portfolio entries — all in one place.

Designed with a clean UI, dark mode support, and persistent data storage.

✨ Features
📊 Dashboard

Overview of all productivity tools

Quick access to each module

💰 Expense Tracker

Track daily expenses

Visual spending chart

Currency displayed in ₦ Nigerian Naira

📚 Study Planner

Add study tasks

Set study dates

Track completed vs pending tasks

💼 Job Tracker

Track job applications

Add:

Company name

Role

Application status

🧑‍💻 Portfolio Builder

Save project ideas

Add:

Project title

Description

Project URL

🌙 Dark Mode

Full page dark mode support

Toggle between light and dark themes

💾 Persistent Storage

Data saved using LocalStorage

Information remains after refreshing the page

🛠 Technologies Used

React

React Router

Recharts (for charts)

CSS3

JavaScript (ES6)

LocalStorage

🚀 Installation

Clone the repository:

git clone https://github.com/Onyinye-Maggie/Productivity-Hub.git

Navigate to the project folder:

cd Productivity-Hub

Install dependencies:

npm install

Run the development server:

npm start
